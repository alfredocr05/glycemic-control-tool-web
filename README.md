# Perioperative Insulin Management Tool

Web-based clinical decision support tool designed to assist with perioperative insulin management across different care settings (Pre-op/DOSA, OR, PACU).

🔗 Live tool: https://alfredocr05.github.io/glycemic-control-tool-web/

---

## Overview

This tool provides a structured approach to perioperative glycemic management by guiding clinicians through key decision points and directing them to the appropriate insulin protocol.

It is designed for use in:
- Pre-operative holding areas (DOSA)
- Intraoperative settings (OR)
- Post-anesthesia care units (PACU)

---

## Features

- Stepwise decision logic for insulin management
- Context-aware protocol selection
- Quick access to:
  - Continuous IV insulin protocols
  - Subcutaneous sliding scale regimens
  - Hypoglycemia management
- Clean and intuitive user interface for rapid clinical use

---

## How It Works

The tool guides the user through two key questions:

1. Patient location:
   - Pre-op / DOSA
   - OR (Intraoperative)
   - PACU / Recovery

2. Insulin status:
   - On continuous IV insulin infusion
   - Not on infusion (subcutaneous management)

Based on these inputs, the tool directs users to the appropriate protocol.

---

## Intended Use

This tool is intended as a **clinical reference aid** for anesthesia providers and perioperative teams.

It supports decision-making but does not replace:
- Clinical judgment
- Institutional protocols
- Specialist consultation when needed
